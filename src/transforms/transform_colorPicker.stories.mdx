# transform_colorPicker

Converts a hexadecimal string to a readonly Color Picker.

#### Conversions

* `"#a8001791"` => <TransformExample cb={transform_colorPicker()} data={"#a8001791"} />

#### Args

None

#### Example

```js
import { transform_colorPicker } from "@simpleview/sv-mosaic"
...
transforms : [
    transform_colorPicker()
]
```
