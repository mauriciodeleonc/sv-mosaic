# transform_chips

Converts an array of MosaicLabelValue elements into Chips.

#### Conversions

* `[{label: "Chip 1", value: "chip_1"}]` => <TransformExample cb={transform_chips()} data={[{label: "Chip 1", value: "chip_1"}]} />

#### Args

None

#### Example

```js
import { transform_chips } from "@simpleview/sv-mosaic"
...
transforms : [
    transform_chips()
]
```
