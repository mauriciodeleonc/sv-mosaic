# transform_dateFormat

Converts a Date object into a month / day / year string.

#### Conversions

* `new Date("December 25, 2023")` => <TransformExample cb={transform_dateFormat()} data={new Date("December 25, 2023")} />

#### Args

None

#### Example

```js
import { transform_dateFormat } from "@simpleview/sv-mosaic"
...
transforms : [
    transform_dateFormat()
]
```
