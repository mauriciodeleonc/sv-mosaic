# FormFieldDropdownMultipleSelection

The `FormFieldDropdownMultipleSelection` component is built over a wrapper for [Material-UI Autocomplete](https://material-ui.com/components/autocomplete/) but with SimpleView brand colors.

- Dropdown with multiple selection and tags visualization
- A tag is a label used to categorize content
