# Icon Button

The `IconButton` component is built over a wrapper for [Material-UI IconButton](https://mui.com/api/icon-button/) but with SimpleView brand colors.

- Icon buttons allow users to take action and make choices with a single tap, e.g. save, delete, add an item, undo, create a new item. 
- Trigger a new element to appear on the page e.g. toggle additional information, open a modal or reveal an overflow menu.
